The ``1-my_list`` module
======================

usinf ``Mylist``
```````````````````

Importing the function from the module
	>>> MyList = __import__('1-my_list').MyList

Checking for all positve list

	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

Checking for all negative numbers list

	>>> my_list = MyList()
	>>> my_list.append(-4)
	>>> my_list.append(-3)
	>>> my_list.append(-2)
	>>> my_list.append(-1)
	>>> my_list.print_sorted()
	[-4, -3, -2, -1]

Checking for some numbers that are negative

	>>> my_list = MyList()
	>>> my_list.append(-1)
	>>> my_list.append(-2)
	>>> my_list.append(3)
	>>> my_list.append(4)
	>>> my_list.append(0)
	>>> my_list.print_sorted()
	[-2, -1, 0, 3, 4]

Checking for Empty List

	>>> my_list = MyList()
	>>> my_list.print_sorted()
	[]

Checking for a single item list

	>>> my_list = MyList()
	>>> my_list.append(4)
	>>> my_list.print_sorted()
	[4]

Checking for a single negative list

	>>> my_list = MyList()
	>>> my_list.append(-4)
	>>> my_list.print_sorted()
	[-4]
